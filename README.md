# meetApp

*****Overview*****
A serverless, progressive web application (PWA) with React using a
test-driven development (TDD) technique. The application uses the Google
Calendar API to fetch upcoming events.
The server is a serverless function hosted by a cloud provider (AWS). The application itself is also hosted online to make it shareable and installable. It can
be used even when the user is offline. As it’s responsive, it displays well on any device.

*****Key Features*****
● Filter events by city.

● Show/hide event details.

● Specify number of events.

● Use the app when offline.

● Add an app shortcut to the home screen.

● View a chart showing the number of upcoming events by city.
